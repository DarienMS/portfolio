Bienvenido a mi Portafolio

Me considero una persona confiable para grandes responsabilidades, puntual, ordenado, predispuesto, amable y respetuoso. Mis Proyectos

Recetas SR

Pagina de recetas desarrollada en Django, utilizando MySQL como base de datos. Permite a los usuarios buscar, filtrar y ver recetas, ademas creamos su propio sistema de administracion el cual esta integrado directame en la pagina al acceder con una cuenta de administrador.

Fernandez Repuestos

Pagina de Repuestos de camion desarrollada para un cliente de mi localidad, una pagina simple para mostrar el negocio y su ubicacion junto a su contacto, Desarrollada solo con HTML, CSS Y JS.

Fast Food App

Aplicacion de comida rapida desarrollada en React Native, con Redux para el manejo de estados y Firebase como base de datos, en la imagen se aprecia el perfil del usuario y la posibilidad de editarlo cambiando direccion o foto de perfil.
